Minecraft 26.1.2 · Fabric

Fish smarter, not harder

Feeshman Deelux adds advanced auto-fishing for Fabric: human-like bite detection, a live HUD, session and lifetime stats, SQLite-backed persistence, and optional ModMenu integration.

Download releases View source

Features

Bite detection

Multi-method detection—velocity, bobber behavior, water checks—with timing that feels natural.

Automation

Auto-cast and reel with randomized delays, stuck handling, and sensible mob avoidance.

HUD

Session counts, rod durability, weather, biome, catch rate, and status at a glance.

Stats & achievements

SQLite-backed stats, leaderboards, commands for history and tops—plus milestone toasts.

Quick install

Current mod version 1.4.0 — requires Fabric Loader and Fabric API versions compatible with your game build.

Minecraft 26.1.2 Loader 0.19.2+ Fabric API 0.148.0+26.1.2 Java 25+ (runtime)
  1. Install Fabric Loader for your Minecraft version.
  2. Add Fabric API to your mods folder.
  3. Drop Feeshman Deelux into mods.
  4. In-game: equip a fishing rod, press O to toggle auto-fishing.

Optional: Mod Menu for in-game config (sound, achievements screen).
